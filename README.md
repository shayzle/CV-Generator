# CV-Generator
This is a project school that I built with HTML, CSS, Bootstrap, JavaScript as the Front-End Part and PHP as the Back-End Part. I built in 2 weeks as a solo school project from Epitech, Paris.


## Prerequisites
PHP 8.2.4


## Clone the repository
git clone : https://github.com/EpitechWebAcademiePromo2027/W-WEB-101-PAR-1-1-generateur_de_cv-8.git   <br>
cd : W-WEB-101-PAR-1-1-generateur_de_cv-8


## Install dependencies (DomPDF)
composer install


## Start the built-in PHP server
php -S localhost:8000


*After this, open the browser and go to* : (http://localhost:8000/cv.php)
